## How to Capture a bill

<br />

* Click on **Capture Bill** to capture a bill

  ![alt text](/images/captureBills.png "Title")

<br />

* select the  **Encounter Type** on the provider details section

<br />

  ![alt text](/images/CaptureBillsModal1.png "Title")

<br />


* input  an  **authorization code** if any
* also type in the first two letter of a diagnosis in the **diagnosis** input and select the diagnosis

<br />

  ![alt text](/images/authorizationCodeifany.png "Title")

<br />

* Scroll down to the  **Service Details** section
* select a service and click on **Add Service**
* The added service shows up in a table below the **Add Service** button

<br />

  ![alt text](/images/serviceDetails.png "Title")

<br />

* thesame goes for the  **Drugs Details** section
* type in a commment in the **Comment** input 
* Click on the **Save Bill** button to save the bill.. The saved bill will now be under the viewed saved bill section
